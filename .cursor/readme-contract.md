# README contract (VisuDEV)

## Living sections

- `README.md` — feature tables, dev commands (`npm run dev`, `dev:hybrid`, `checks`, `push`)
- `README.md` — `## Recent changes` (max 10 Zeilen, neueste zuerst)
- `docs/HYBRID_DEV.md` — lokaler Supabase + Hybrid-Workflow
- `docs/SUPABASE_SETUP.md` — Cloud/lokal, Env, Deploy
- `AGENTS.md` — nur bei Änderungen am Agent-/Check-Workflow

## Skip README

Nur Tests, reines Formatting, Lockfiles ohne Verhaltensänderung.

## Recent changes format

`- **YYYY-MM-DD** — Kurzbeschreibung (\`branch\` oder Issue #n)`
